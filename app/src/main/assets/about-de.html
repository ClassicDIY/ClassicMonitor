Classic-Monitor ist ein Statusmonitor für die Midnite Solar's Classic 150, 200 und 250 Laderegler (www.midniteSolar.com).

Das Programm liest die Daten nur aus.
Die Software wird so wie sie ist bereitgestellt, ohne irgendeine Garantie, weder ausdrücklich noch stillschweigend.
Classic-Monitor ist kein Produkt von Midnite, auch keine Unterstützung seitens Midnite.

Erkennt die App einen aktuellen Whizbang Junior Strommonitor, erscheint der Status in der bi-direktionalen Stomanzeige.
Unterstützung für die Tristar MPPT Laderegler von Morningstar ist ebenfalls in den Grundfunktionen enthalten.
Aufgezeichneten Protokolle können an pvoutput.org weitergeleitet werden, eine API-Key und SID ist in den Einstellungen für jeden Laderegler anzugeben.

Diese Anwendung verwendet die folgenden Open-Source-Komponenten:

  • J2MOD Modbus-Bibliothek, erstellt von Julie Haugh: SourceForge.
  • Autofittextview, erstellt von Grantland Chew Bibliothek: github.com/grantland.
  • Joda-time Bibliothek, erstellt von Stephen Colebourne: www.joda.org/joda-time.
  • Google-gson google-gson.
  • MPAndroidChart, erstellt von Philipp Jahoda located at github.com/PhilJay/MPAndroidChart.
