Classic Monitor (version 3.0.7)

Si vous trouvez cette application utile, veuillez aider à soutenir le développement!

Le code est disponible sur github

Classic Monitor est un moniteur de surveillance pour le Contrôleur de charge Midnite Solar Classic 150, 200, 250 (www.midniteSolar.com).

Ce est un programme en lecture seule, il n'écrire pas à le Contrôleur.
Le logiciel est fourni "TEL QUEL", SANS GARANTIE D'AUCUNE SORTE, expresse ou implicite.
Classic Monitor n'est pas un produit de Midnite Solar, ni soutient-elle cette application.

Si l'application détecte un moniteur actuel Whizbang Junior, l'état de charge se affiche avec une jauge actuelle bidirectionnel.
Support de base pour le contrôleur de charge Tristar MPPT de Morningstar est également inclus.
L'énergie enregistrée quotidiens peuvent être transmis à pvoutput.org en fournissant une clé API et SID # dans les paramètres pour chaque contrôleur.

Cette application utilise les composants open source suivants.

  • La librairie J2MOD modbus créé par Julie Haugh situé à SourceForge.
  • La librairie Autofittextview créé par Grantland Chew situé à github.com/grantland.
  • La librairie Joda-time créé par Stephen Colebourne situé à www.joda.org/joda-time.
  • La librairie Google-gson situé à google-gson.
  • La librairie MPAndroidChart créé par Philipp Jahoda situé à github.com/PhilJay/MPAndroidChart.
