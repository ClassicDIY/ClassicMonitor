Classic Monitor (versione 3.0.7)

Se trovi utile questa app, ti preghiamo di supportarne lo sviluppo!

Il suo codice sorgente è ospitato e accessibile a github

Classic Monitor è un monitor di stato per Midnite solar Classic 150, 200, 250 regolatore di carica (www.midniteSolar.com).

Si tratta di un solo programma di lettura, esso non scrivere la Classic.
Il software viene fornito "COSÌ COM'È", SENZA ALCUN TIPO DI GARANZIA, esplicita o implicita.
Monitor Classic non è un prodotto di Midnite solare, né supporta questa applicazione.

Se l'applicazione rileva un monitor corrente Whizbang Junior, lo stato di carica viene visualizzata con un misuratore di corrente bidirezionale.
Il supporto di base per il regolatore di carica Tristar MPPT da Morningstar è anche incluso.
I registri di energia registrato quotidiane possono essere inoltrate a pvoutput.org fornendo una chiave API e SID# nelle impostazioni per ciascun controller.

Questa applicazione utilizza i seguenti componenti open source.

  • Biblioteca Modbus J2MOD creato da Julie Haugh situato a SourceForge.
  • Autofittextview biblioteca creato da Grantland Chew trova github.com/grantland.
  • Biblioteca Joda-tempo creato da Stephen Colebourne situato a www.joda.org/joda-time.
  • Biblioteca Google-GSON situato a google-gson.
  • Biblioteca MPAndroidChart creato da Philipp Jahoda trova github.com/PhilJay/MPAndroidChart.

Questa applicazione è open source e rilasciato sotto la licenza Apache V2.0.
